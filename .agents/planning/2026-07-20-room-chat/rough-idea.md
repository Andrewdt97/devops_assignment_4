# Rough Idea

Design a live, room-based chat for this poker game.

## Context

This is for the existing Multiplayer Poker application — a real-time multiplayer Texas Hold'em poker game built with Node.js, Express, and WebSockets. Players create/join rooms with a 6-character room code and play with 2–8 players per table. The game already uses WebSockets for real-time game state communication.

The chat feature should allow players in the same room to communicate with each other in real-time during gameplay.
